# Olden Era RMG Preview

Static browser-based reviewer for Olden Era `.rmg.json` map templates.

## Use

Open the GitHub Pages site, click `Load .rmg.json`, and select a local template file.

The app runs fully in the browser. Template files are not uploaded anywhere.

## GitHub Pages

This repository deploys from `master` using GitHub Actions. After pushing to
`git@github.com:golem15com/oldenera-rmg-preview.git`, enable Pages in the
repository settings with source `GitHub Actions` if it is not already enabled.
